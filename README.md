**❓ What is this?** Extension enhancing Librus Synergia: grade averages and additional features that improve the use of the e-diary.

**❓ How to install it?**
* [**Install from Chrome Web Store**](#-install-from-chrome-web-store)
* [**Install from releases**](#-install-from-releases)

**❓ What did I use?**
* [Coding](https://code.visualstudio.com/)
* [Git management](https://desktop.github.com/)

## 📥 Install from Chrome Web Store
1) Navigate to [extension page on Chrome Web Store](https://chromewebstore.google.com/detail/fcklhekibdibjkphlllegpkagaedkhnp?utm_source=github-repo)
2) Click on button adding extension
3) Navigate to Librus Synergia and start using it!

## 📦 Install from releases
1) Navigate to [releases](https://github.com/jbakalarski/BetterLibrus/releases)
2) Download `Better Librus v{version tag}.zip` from latest version
3) Unzip it
4) Navigate to extension settings of your browser
5) Click on `Add unpacked`
6) Choose directory of unzipped extension
7) Navigate to Librus Synergia and start using it!

## 🚀 Features
* Displays calculated grade averages in the grades view
* Supports both standard grades and point-based grading
* Improves data readability without altering the behavior of Librus itself
